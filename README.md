# 介绍
这是一个专门给经典后台管理站点准备的前端钩子(hooks)，主要帮助开发人员可以快速的渲染列表、弹出框等UI元素，力求减少开发人员不必要的重复代码，提高效率。**目前只支持Vue3。**

# 有什么特点
- 不依赖控件库
- ts实现，很好的代码提示
- 示例丰富，支持主流的控件库。例如：tdesign, element-ui, ant design等
- 图表支持echart

# 支持的UI元素
- Table 列表页
- Modal 模态框，包含Dialog Drawer
- Chart 图表

# 运行
## 安装
```bash
yarn add @2kk/admin-vueuse
```

## 本地运行
```bash
yarn

yarn build

yarn dev
```
